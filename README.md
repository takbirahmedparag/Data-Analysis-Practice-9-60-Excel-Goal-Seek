# Data-Analysis-Practice-9-60-Excel-Goal-Seek
This repository contains my Excel practice on Goal Seek. In this project, I used the Goal Seek feature to determine the required price per unit to achieve a target profit. Based on quantity sold, Excel automatically adjusts the price value to reach the desired profit. This practice is part of my Data Analysis Practice Series (9/60).

## Project Objective
The main objective of this project is to learn how Goal Seek works in Excel to find an unknown value based on a desired result.
This practice helps to:
•	Understand the Goal Seek feature
•	Perform what-if analysis
•	Find required values to achieve a target profit
•	Improve Excel business analysis skills

## Tasks Covered
Use Goal Seek to determine the required price per unit when Quantity Sold = 50 units and the target profit = 15,000 Taka.
Adjust the Price per Unit automatically so that the Profit value becomes 15,000 Taka

## KPI (Key Performance Indicators)
•	Correct use of Goal Seek
•	Accurate target profit calculation
•	Proper what-if analysis
•	Correct adjustment of price per unit

## Process
1.	Created a dataset including Price, Quantity, and Profit.
2.	Used the formula Profit = Price × Quantity.
3.	Applied Goal Seek from the What-If Analysis tool.
4.	Set the Profit cell to 15,000.
5.	Allowed Excel to change the Price cell automatically to reach the target value.
